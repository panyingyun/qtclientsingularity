windows编译环境：
QT5.12.2
visual studio 2017

创建下面环境变量（自动创建就不需要创建了）：
QT_PATH  C:\Qt\Qt5.12.2\5.12.2\msvc2017_64
Qt5_DIR    C:\Qt\Qt5.12.2\5.12.2\msvc2017_64\lib\cmake\Qt5

添加环境变量
PATH中添加 C:\Qt\Qt5.12.2\5.12.2\msvc2017_64\bin


Linux 环境
sudo apt-get install libgl1-mesa-dev  libx11-dev
sudo apt-get install qt5-default  qtcreator
